# Movie Library

## Автор
**Никифоренко Андрей Иванович**

## Описание
Простое десктопное приложение для хранения и управления личной коллекцией фильмов.

### Возможности
- Добавление фильмов (название, жанр, год, рейтинг)
- Просмотр в удобной таблице
- Фильтрация по жанру и году
- Сохранение данных в `movies.json`
- Валидация вводимых данных
- Удаление фильмов

## Как запустить
1. Установите Python 3.8+
2. Запустите файл:
   ```bash
   python movie_library.py
